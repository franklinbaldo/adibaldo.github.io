# Site do seu Adi (GitHub Pages)

Este repositório publica o site em **https://adibaldo.github.io/**.

## Como publicar
- Para criar um post novo: adicione um arquivo em `src/content/blog/`.
- Depois é só dar *commit* e *push* que o site atualiza.

## Feito com
- Astro (template blog)
- Deploy via GitHub Actions
